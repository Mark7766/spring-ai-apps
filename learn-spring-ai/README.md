## 📖 系列介绍

欢迎来到 **Spring AI 应用开发实战系列**！

这是一套完整的Spring AI学习教程，通过 **15期连载文章**（约156,000字，230+代码示例），从零基础到企业级部署，系统讲解Spring AI应用开发全技术栈。每一期都聚焦一个实际开发问题，提供理论讲解和可运行的示例代码。

### 🎯 为什么选择这个系列？

- ✅ **系统完整**：15期涵盖Spring AI全技术栈，156,000字深度讲解
- ✅ **问题导向**：每期解决一个实际开发痛点，直击要害
- ✅ **代码可用**：230+个代码示例，12个完整项目，开箱即用
- ✅ **循序渐进**：从Hello World到企业级部署，环环相扣
- ✅ **已完结**：全系列15期已完成，可系统学习

### 🎓 学习目标

完成本系列后，你将能够：

```
✅ 独立搭建Spring AI开发环境
✅ 快速接入主流AI模型服务（OpenAI、Azure、Ollama等）
✅ 实现企业级RAG问答系统
✅ 开发具备工具调用能力的AI Agent
✅ 设计高可用的AI应用架构
✅ 将AI应用部署到生产环境（Docker + K8s）
```

## 🚀 快速开始

### 环境要求

```
- JDK: 17+
- Maven: 3.6+
- IDE: IntelliJ IDEA（推荐）
- 系统: Windows/macOS/Linux
```

### 克隆项目

```bash
# 克隆完整项目
git clone https://github.com/Mark7766/spring-ai-apps.git
cd spring-ai-apps

# 进入文档目录
cd learn-spring-ai
```

### 三步开始学习

```
1️⃣ 阅读系列规划
   └─ 查看 00-规划/系列规划.md 了解整体学习路径

2️⃣ 从第1期开始
   └─ 阅读 01-入门篇/01-开启AI应用开发之旅.md
   └─ 运行示例项目 chat-deepseek

3️⃣ 循序渐进学习
   └─ 按顺序完成15期文章
   └─ 完成每期的思考题
   └─ 运行所有示例项目
```

```

   ↓

   └─ Embedding、向量库、文档处理、知识图谱

   ↓
   掌握高级特性
   └─ Function Calling、Text-to-SQL、模板生成、MCP

   ↓
   形成知识体系

```

### 学习建议

| 建议 | 说明 |
|------|------|
| **1. 循序渐进** | 按顺序学习，不要跳过基础内容 |

| **3. 记录笔记** | 整理遇到的问题和解决方案 |
| **4. 思考总结** | 完成每期的思考题，加深理解 |
| **5. 社区交流** | 在Issues中提问和分享经验 |

